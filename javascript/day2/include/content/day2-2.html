let과 var의 차이

  • if(true) {
    var x = 5;
    }
    console.log(x); <-- 5
  • if(ture) {
    let y = 5;
    }
    console.log(y); <-- ReferenceError : y is not defined
  • x의 경우 변수를 var로 선언했기 때문에 x의 스코프가 전역 맥락이기 때문에 x의 스코프는 if문 블록에 제한되지 않는다.
  • y의 경우 let으로 변수를 선언했기 때문에 if문 블록에 한해서 제한이 된다.
