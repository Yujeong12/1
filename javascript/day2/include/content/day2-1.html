자바에서의 변수 선언

var firstVar = 10;

  • 자바스크립트는 대소문자를 구분할 줄 안다.
  • 유니코드 문자셋을 이용
  • _로 띄어쓰기를 구분하는 방식과, 대문자를 쓰는 방식이 있지만,
    자바스크립트는 관행적으로 대소문자 방식을 더 많이 이용.
선언 방법
  • var : 변수를 선언, 추가로 동시에 값을 초기화.
  • let : 블록 스코프 지역 변수를 선언. 추가로 동시에 값을 초기화
  • cosnt : 블록스코프 읽기 전용 상수를 선언.
  • 자바스크립트는 문자, 숫자, _혹은 &로 식별자 이름을 시작해야한다.
  • var 키워드로 변수는 맥락에 따라 지역 및 전역 변수를 선언하는데 모두 사용.
    let혹은 const의 경우 블록 스코프 지역 변수를 선언하는데 사용.
  • var a;
    console.log(a) <-- undefined

    console.log(b) <-- undefined
    var b;

    console.log(c) <-- Uncaught ReferenceError : c is not defined

    let x;
    console.log(x) <-- undefined

    console.log(y); <-- Uncaught ReferenceError : y is not defined
    let y;
