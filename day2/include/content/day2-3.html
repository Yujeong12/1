호이스팅

아까 위의 변수 선언 부분의 예시 코드에서 보았다시피, 자바스크립트는 나중에 선언한 변수를 참조할 수 있다.(변수 호이스팅)
함수에서는 함수 선언으로는 호이스팅이 되지만 함수 표현식으로는 호이스팅이 되지 않는다.

  • 함수 선언
    foo(); <-- "bar"
    function foo() {
    console.log('bar');
    }

    함수 표현식
    baz(); <-- TypeError : baz is not a function
    var baz = funtion() {
    console.log('bar2');
    };
